# IT-product-creating

## BMI Calculator (консольный)

Скрипт запрашивает вес в кг и рост в метрах, вычисляет BMI по формуле:

\[
BMI = \frac{вес}{рост^2}
\]

Далее выводит значение BMI и категорию здоровья: **недовес**, **нормальный**, **избыточный**, **ожирение**.

## Запуск

Из корня проекта:

```bash
python bmi_calculator.py
```

Примечания:
- Ввод дробных чисел поддерживает и точку, и запятую (например `1.75` или `1,75`).
- Рост вводится **в метрах** (например `1.80`), вес — **в килограммах**.
